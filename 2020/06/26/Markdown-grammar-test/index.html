Markdown 语法测试

自用的 cheat sheet 😉

一、字体

  1. 斜体：*italic*，效果：italic，中文不使用斜体；
  2. 粗体：**bold 粗体**，效果：bold 粗体；
  3. 粗斜体：***bold italic***，效果：bold italic，尽量不用；
  4. 删除线：~~delete 删除文字~~，效果：delete 删除文字；
  5. 下划线：++下划线++，效果：下划线；
  6. 高亮文字：==高亮文字==，效果：高亮文字；
  7. 文字颜色（使用 HTML 语句）：
    1. 标准色：以红色为例：<font color=red> 红色文字 </font>，效果： 红色文字 ；
    2. 任意颜色：<font color=#b100b1> 任意颜色文字 </font>，效果： 任意颜色文字 ；
  8. 字号（使用 HTML 语句）：
    1. 缩小：<font size=1> 字号小一些 </font>，效果： 字号小一些 ；
    2. 放大：<font size=5> 字号大一些 </font>，效果： 字号大一些 ；
  9. 上标：5 m^2^，效果：5 m2；
  10. 下标：CO~2~，效果：CO2。

二、段落

  1. 标题（效果略）：
    1. 二级标题：## 二级标题示例，效果略；
    2. 三级标题：### 三级标题示例，效果略；
  2. 整行横线：***，必须在一行的起始处使用，效果略；
  3. 空行、换行（使用 HTML 语句）：<br />新的一行，效果：
    新的一行；
  4. 对齐（使用 HTML 语句）：
    1. 左对齐：<p align="left"> 左对齐 </p>，效果：

      左对齐

    2. 居中对齐：<center> 居中对齐 </center>，效果：
      居中对齐
    3. 右对齐：<p align="right"> 右对齐 </p>，效果：

      右对齐

  5. 首行缩进 2 字符：在段首添加 2 个 em 空格或 4 个 en 空格：
    1. 2 个 em 空格：&#8195;&#8195;首行缩进，效果：  首行缩进；
    2. 4 个 en 空格：&#8194;&#8194;&#8194;&#8194;首行缩进，效果：    首行缩进。

三、特殊符号

  1. 连字符 / hyphen：-，效果：-；
  2. 短连接号 / en dash：--，效果：–；
  3. 长连接号 / en dash：---，效果：—；
  4. 方括号：\[\]，效果：[]；
  5. 特殊符号[1] ：&#233 &#952 &#8704，效果：é θ ∀；
  6. Emoji[2] ：:grimacing: :tropical_fish: :rocket:，效果：😬 🐠 🚀；

五、插入环境

5.1 数学环境

5.1.1 行内数学公式

置于一对$中，其余和 LaTeX 语法相同，注意代码中的某些地方可能需要添加额外的空格以区分各命令。示例：$V=\frac{4}{3}\pi R^3$

5.1.2 行间数学公式

置于\begin{equation}和\end{equation}中，其余和 LaTeX 语法相同，注意代码中的某些地方可能需要添加额外的空格以区分各命令。目前发现小空格转义字符\,无法正常编译。示例：
\begin{equation}
U_M = \sqrt{\sum_{i=1}^m \left(\frac{\partial f}{\partial x_i}\cdot U_{xi}\right)^2}
\end{equation}
不带编号的行间公式置于\begin{equation*}和\end{equation*}中即可。
